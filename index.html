  • Free

    Our game is free forever. We are an open source clone of the Minecraft (Mojang/Microsoft)

  • Minecraft's JAVA MOD's native support

    In our game you can play with Minecraft java mod's immediately after installing them on your computer.

  • Open source

    Our code is open for all people and all, that created in our game or for our game (maps, mod's, textures, plugin's, etc.) have license GPL 3.0 and higher.

  • Mincraft's JAVA server's native support

    In our game you can play on Minecraft java servers immediately after installing.

  • Open API

    Our API is open for all developers, who want create mods, plugin's and textures for our game

  • Crossplatform

    You can install our game on all PC systems.And we think about ios and android versions
